It is unclear as to which Horus name _Weneg_ correspond. Egyptologists match the name to Nebra or Sekhemib, but it might also be an independent ruler. The New Kingdom name Wadjenes and Wadjelas. According to Africanus, Manetho called him _Tlas_, which in itself probably is based on the coptic rewriting of Wadjenes as _Ugotlas_.
