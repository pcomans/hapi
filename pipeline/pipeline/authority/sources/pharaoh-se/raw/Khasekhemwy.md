# Khasekhemwy in hieroglyphs

The eleventh pharaoh of the [Second Dynasty](https://pharaoh.se/ancient-egypt/dynasty/2), _a.k.a. Khasekhemui_

| Precedessor<br>[Hudjefa I](https://pharaoh.se/ancient-egypt/pharaoh/Hudjefa-I)<br>Successor<br>[Nebka](https://pharaoh.se/ancient-egypt/pharaoh/Nebka) |
| Reign of **Khasekhemwy** |
| AE Chronology | 2610–2593 |
| von Beckerath | 2709–2682 |
| Malek | 2674–2647 |
| Redford | 2714–2687 |
| Turin King List | 27y 2m 1d |
| Manetho | 30 years |

The Horus name Khasekhem is the only one that appears with the falcon, the other two variants both appear with the falcon and the Seth-animal. The New Kingdom king lists have different names, which probably disagree because the scribes misinterpreted the similar signs in the unknown sources.

## Horus names

![Horus name](https://pharaoh.se/svg/hiero/t_horus.svg)

The oldest form of the name of a pharaoh, a falcon representing Horus is perched on the palace facade.

Horus name

## Nebty names

![Nebty name](https://pharaoh.se/svg/hiero/t_nebty.svg)

The “Two Ladies”, the _goddesses Nekhbet and Wadjet_, representing Upper and Lower Egypt respectively.

## Throne names

![Prenomen or throne name](https://pharaoh.se/svg/hiero/t_nesut.svg)
