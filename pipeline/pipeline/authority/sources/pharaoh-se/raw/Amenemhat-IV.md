# Amenemhat IV in hieroglyphs

The seventh pharaoh of the [Twelfth Dynasty](https://pharaoh.se/ancient-egypt/dynasty/12), _a.k.a. Amenemhet IV_

| Precedessor<br>[Amenemhat III](https://pharaoh.se/ancient-egypt/pharaoh/Amenemhat-III)<br>Successor<br>[Neferusobek](https://pharaoh.se/ancient-egypt/pharaoh/Neferusobek) |

| AE Chronology | 1772–1764 |
| von Beckerath | 1807–1798 |
| Shaw | 1786–1777 |
| Dodson | 1798–1785 |
| Arnold | 1799–1787 |
| Malek | 1814–1805 |
| Grimal | 1797–1790 |
| Franke | 1773–1764 |
| Redford | 1798–1790 |
| Turin King List | 9y 3m 27d |

His throne name is present on all four of the New Kingdom king lists, and the Turin Canon preserves his length of reign as 9 years, 3 months and 27 days. His name is not present in the king list of Manetho.

## Nebty name

![Nebty name](https://pharaoh.se/svg/hiero/t_nebty.svg)

The “Two Ladies”, the _goddesses Nekhbet and Wadjet_, representing Upper and Lower Egypt respectively.

## Throne names

![Prenomen or throne name](https://pharaoh.se/svg/hiero/t_nesut.svg)

Announced at the coronation and always written in a cartouche. The “official” name of the pharaoh. Also known as the _Prenomen_.

Throne name

![Maa kheru Ra](https://pharaoh.se/svg/pharaoh/12-07-04.svg)
