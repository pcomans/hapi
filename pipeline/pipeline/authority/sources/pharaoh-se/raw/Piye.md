# Piye in hieroglyphs

The first pharaoh of the [Twenty-fifth Dynasty](https://pharaoh.se/ancient-egypt/dynasty/25), _a.k.a. Piankhi, Piankhy, Pije, Pianchy_

| Precedessor<br>[Bakenrenef](https://pharaoh.se/ancient-egypt/pharaoh/Bakenrenef)<br>Successor<br>[Shabaka](https://pharaoh.se/ancient-egypt/pharaoh/Shabaka) |
| Reign of **Piye** |
| AE Chronology | 753–723 |
| von Beckerath | 746–713 |
| Shaw | 747–716 |
| Dodson | 752–717 |
| Arnold | 747–716 |
| Grimal | 747–716 |
| Redford | 735–712 |

Piye was king of the Kingdom of Kush but also ruled Upper Egypt which had been peacefully integrated as Egypt itself was in full decline and torn by power struggles among local petty rulers. He slowly expanded north and conquered towns one by one until Memphis was captured.

## Throne names

![Prenomen or throne name](https://pharaoh.se/svg/hiero/t_nesut.svg)
