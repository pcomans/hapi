# Sneferka in hieroglyphs

The eighth pharaoh of the [Second Dynasty](https://pharaoh.se/ancient-egypt/dynasty/2), _a.k.a. Sekanefer, Neferseka, Seneferka_

| Precedessor<br>[Peribsen](https://pharaoh.se/ancient-egypt/pharaoh/Peribsen)<br>Successor<br>[Neferkasokar](https://pharaoh.se/ancient-egypt/pharaoh/Neferkasokar) |
| Reign of **Sneferka** |
| Malek | 2690–2682 |

The New Kingdom scribes "corrected" the name **Sneferka** into **Neferkara** by exchanging the initial **S** with the honorific **Ra**. This was a common practice that made sense to the scribes, as can be noted several times in the king lists.
