# Qaa in hieroglyphs

The eighth pharaoh of the [First Dynasty](https://pharaoh.se/ancient-egypt/dynasty/1), _a.k.a. Kaa_

| Precedessor<br>[Semerkhet](https://pharaoh.se/ancient-egypt/pharaoh/Semerkhet)<br>Successor<br>[Hotepsekhemwy](https://pharaoh.se/ancient-egypt/pharaoh/Hotepsekhemwy) |
| Reign of **Qaa** |
| AE Chronology | 2775–2732 |
| von Beckerath | 2853–2828 |
| Malek | 2818–2793 |
| Grimal | 2960–2926 |
| Turin King List | 95 (years?) |
| Manetho | 26 years |

The Horus name Qa'a is well attested, and according to Africanus, Manetho named him Bieneches, but according to Eusebius it was Ubienthes. The New Kingdom king lists name him as Qebeh or Qebehu.

## Throne names

![Prenomen or throne name](https://pharaoh.se/svg/hiero/t_nesut.svg)

Announced at the coronation and always written in a cartouche. The “official” name of the pharaoh. Also known as the _Prenomen_.
