# Neferkara Tereru in hieroglyphs

The tenth pharaoh of the [Eighth Dynasty](https://pharaoh.se/ancient-egypt/dynasty/8), _a.k.a. Neferkare Tereru_

| Precedessor<br>[Nikara](https://pharaoh.se/ancient-egypt/pharaoh/Nikara)<br>Successor<br>[Neferkahor](https://pharaoh.se/ancient-egypt/pharaoh/Neferkahor) |
| Reign of **Neferkara Tereru** |

Neferkara Tereru is found on the Abydos king list number 49, while the Turin papyrus has a notation of a lacuna where the name would have been found.
